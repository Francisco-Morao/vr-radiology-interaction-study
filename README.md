# VR Radiological Interaction Study

This repository contains the Unity project used in the user study
comparing controller-based and hand-gesture interaction techniques
for 3D object manipulation in virtual reality.

## Requirements
- Unity 6000.0.62f1
- XR Interaction Toolkit
- Compatible VR headset with controller and hand tracking support

## How to Run
1. Open the project in Unity Hub
2. Load the main scene
3. Press Play with a connected VR headset
